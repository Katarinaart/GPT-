`````python

`````

#### **ШАГ 3: Конвертация Markdown → HTML**
`````python
# Применить существующие CSS классы:

# Таблицы:
| Header | Header |     →     <div class="table-container">
|--------|--------|               <table>
| Cell   | Cell   |                 <thead><tr><th>Header</th></tr></thead>
                                   <tbody><tr><td>Cell</td></tr></tbody>
                                 </table>
                               </div>

# Code blocks:
````python               →     <div class="code-block">
code here                     code here
```                           </div>

# Info boxes:
> **💡 Совет**          →     <div class="info-box">
> Текст                        <h4>💡 Совет</h4>
                               <p>Текст</p>
                             </div>

# Заголовки:
## 3.1. Заголовок       →     <h2 id="section3-1">3.1. Заголовок</h2>

# Параграфы:
Текст                   →     <p>Текст</p>

# Списки:
- Item                  →     <ul><li>Item</li></ul>
1. Item                 →     <ol><li>Item</li></ol>
```

#### **ШАГ 4: Интеграция в HTML**
```python
# 1. Прочитать index_new_brain.html
html = read_file('index_new_brain.html')

# 2. Найти позицию вставки (перед </main>)
insert_pos = html.find('</main>')

# 3. Собрать новый контент
new_sections = ""
for i in range(3, 9):  # sections 3-8
    section_html = f'<section class="section" id="section{i}">\n'
    section_html += convert_markdown_to_html(f'section{i}.txt')
    section_html += '</section>\n\n'
    new_sections += section_html

# 4. Вставить
html_new = html[:insert_pos] + new_sections + html[insert_pos:]

# 5. Обновить sidebar navigation
sidebar_nav = generate_sidebar_nav()  # 6 новых nav-group
sidebar_pos = html_new.find('</aside>')
html_new = html_new[:sidebar_pos] + sidebar_nav + html_new[sidebar_pos:]
```
